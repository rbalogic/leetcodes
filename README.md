## 🛠 Usage

To run any of the solutions locally, ensure you have [Node.js](https://nodejs.org/) installed, then execute the script using the following command:

```bash
node <filename>.js
```

Note: Most files contain function definitions as per LeetCode's format. You may need to add console logs or test cases at the bottom of the files to see output in your terminal.
